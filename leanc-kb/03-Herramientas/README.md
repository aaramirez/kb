# Herramientas Lean Change

Las herramientas de Lean Change son prácticas prácticas diseñadas para gestionar el cambio organizacional de manera ágil y centrada en las personas. Cada herramienta proporciona un enfoque estructurado para comprender, diseñar e implementar cambios de forma iterativa y basada en evidencia.

## Cómo usar estas herramientas

1. **Selecciona según la fase** — No todas las herramientas se aplican al mismo tiempo; elige las adecuadas para tu etapa del proceso de cambio.
2. **Combina y adapta** — Las herramientas son modulares; combínalas para crear flujos de trabajo que se ajusten a tu contexto.
3. **Itera con frecuencia** — Usa los ciclos de retroalimentación para ajustar el enfoque según los resultados obtenidos.
4. **Involucra al equipo** — La mayoría de estas herramientas están diseñadas para trabajo colaborativo; úsalas con tu equipo y stakeholders.
