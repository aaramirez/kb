## Filosofía de Selección

- Elegir herramientas que se integren entre sí
- Preferir soluciones con ecosistemas activos
- Evaluar costo total de propiedad, no solo licencia
- Considerar learning curve del equipo
- Validar con [[14-evaluacion-herramientas|PoC antes de adoptar]]
