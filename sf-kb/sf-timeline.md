# Línea del Tiempo de Software Factory

Evolution de las Software Factories desde los mainframes hasta la era de la IA.

## 1950s — Mainframes
- **1956**: IBM presenta el IBM 305 RAMAC (primer disco duro)
- Programación en ensamblador, sin metodologías formales
- Equipos pequeños, desarrollo artesanal

## 1960s — La Crisis del Software
- **1968**: Primera Conferencia de la OTAN acuña "Ingeniería de Software"
- Problema de la "crisis del software": proyectos retrasados y sobre presupuesto
- Surgimiento de metodologías estructuradas

## 1970s — Metodologías Estructuradas
- **1970**: Modelo Waterfall (Winston Royce)
- **1975**: Brooks escribe "The Mythical Man-Month"
- **1979**: Winston Royce publica el modelo en cascada
- Enfoque en documentación y planificación

## 1980s — Calidad y Procesos
- **1984**: Deming y el control estadístico de procesos
- **1986**: Taguchi y robustez del diseño
- **1987**: ISO 9000 publicado
- Surgimiento de CMM (Capability Maturity Model)

## 1990s — Evolución de Metodologías
- **1991**: Scrum formalizado por Schwaber y Sutherland
- **1993**: Rational Unified Process (RUP)
- **1994**: Extreme Programming (Kent Beck)
- **1995**: Kanban aplicado a software (David Anderson)
- **1996**: Java y el boom de internet
- **1997**: CMMI publicado por SEI

## 2001 — El Manifiesto Ágil
- **2001**: 17 profesores se reúnen en Snowbird, Utah
- Se publica el Manifiesto Ágil
- Scrum, XP, DSDM, Crystal, FDD se unen bajo Agile

## 2000s — Agile y DevOps
- **2003**: The Art of Software Testing (Glenford Myers)
- **2004**: Patrick Debois acuña "DevOps"
- **2008**: Kanban para software (David Anderson)
- **2009**: Primera Conferencia DevOps Days (Gante, Bélgica)
- **2010**: The Phoenix Project (Gene Kim)
- **2011**: Puppet, Chef, Ansible (IaC)

## 2010s — Cloud y Contenedores
- **2013**: Docker publicado
- **2014**: Kubernetes (Google)
- **2015**: Angular 2, React (ecosistema frontend moderno)
- **2016**: GitHub Actions, GitLab CI
- **2017**: Serverless Framework
- **2018**: GitOps definido por Weaveworks
- **2019**: Istio Service Mesh

## 2020s — La Era de la IA
- **2020**: GitHub Copilot (Technical Preview)
- **2021**: GitHub Copilot GA
- **2022**: ChatGPT publicado (OpenAI)
- **2022**: GitHub Copilot Chat
- **2023**: GPT-4, Claude, Gemini — LLMs multimodales
- **2023**: Cursor, Windsurf — IDEs con IA nativa
- **2024**: Model Context Protocol (MCP) — Anthropic
- **2024**: Agentes de IA para código (Devin, OpenHands, SWE-Agent)
- **2024**: Multi-agent systems para desarrollo
- **2025**: AI-native development platforms
- **2025**: Autonomous coding agents
- **2026**: AI en todo el ciclo de vida del software

## 2030+ — El Futuro
- Agentes completamente autónomos
- Software auto-remediante
- Diseño asistido por IA
- Testing autónomo
- Operaciones autónomas (AIOps completo)
