## Definición (WHO)

La OMS clasifica el burnout como "phenomenon occupational" (no disease) caracterizado por tres dimensiones:

| Dimensión | Descripción |
|-----------|-------------|
| **Agotamiento** | Energía mental y física agotada |
| **Cinismo / Desconexión** | Distanciamento mental del trabajo |
| **Reducida eficacia** | Sentimiento de incompetencia y lack de logro |

> **Importante**: Burnout es un problema organizacional, no individual. La culpa no es de la persona.

## Causas de Burnout en Engineers

| Causa organizacional | Ejemplo concreto |
|---------------------|------------------|
| **Workload insostenible** | Deadlines imposibles, crunch persistente |
| **Lack de control** | No tener input en decisiones técnicas |
| **Insuficiente reward** | Recognition escaso, compensation stagnation |
| **Community breakdown** | Silos, lack de support, toxic culture |
| **Absence de fairness** | Bias en promotions, decisions opacas |
| **Values mismatch** | Trabajo contradice valores personales |

### Burnout en Tech (Specific)

| Factor tech | Por qué causa burnout |
|-------------|---------------------|
| **Always-on expectations** | Slack notifications 24/7, on-call rotas |
| **Rapid context switching** | Too many projects, too many tools |
| **Imposter syndrome** | Constant learning pressure |
| **Feature factory pressure** | Velocity over quality |
| **Incident fatigue** | Too many incidents, no root cause fix |

## Sensores de Detección

### Individuales (Self-check)

| Señal | Severidad |
|-------|-----------|
| Difficulty concentrating | 🟡 Early |
| Cynicism about work | 🟡 Early |
| Physical exhaustion | 🟠 Medium |
| Sleep problems | 🟠 Medium |
| Detachment from colleagues | 🔴 Advanced |
| Ineffectiveness at work | 🔴 Advanced |
| Health problems | 🔴 Advanced |

### Organizacionales (Team-level)

| Metric | Red flag |
|--------|----------|
| **eNPS trend** | Declining 2+ quarters |
| **Sick days** | Increasing trend |
| **Voluntary attrition** | Above 15% annually |
| **Overtime hours** | Consistently > 10h/week |
| **On-call incidents** | > 2 incidents/week/person |
| **Meeting load** | > 25h/week |

## Organizational Prevention Strategies

### Workload Management

| Strategy | Implementation |
|----------|---------------|
| **Sustainable pace** | 40h/week standard, no crunch as norm |
| **Capacity planning** | 80% capacity, 20% buffer |
| **Workload visibility** | Team boards showing individual load |
| **Saying no** | Empowered to push back on unreasonable deadlines |

### Autonomy & Control

| Strategy | Implementation |
|----------|---------------|
| **Technical decision-making** | Engineers choose their tools |
| **Flexible schedule** | Async-first, flexible hours |
| **Project selection** | Input into what they work on |
| **Learning time** | Dedicated time for growth |

### Recognition & Growth

| Strategy | Implementation |
|----------|---------------|
| **Regular feedback** | Quarterly career conversations |
| **Public recognition** | Shout-outs in all-hands |
| **Growth opportunities** | Stretch projects, conference attendance |
| **Fair compensation** | Regular market adjustments |

### Community Building

| Strategy | Implementation |
|----------|---------------|
| **Team rituals** | Social time, non-work bonding |
| **Psychological safety** | Blameless culture |
| **Mentoring programs** | Support networks |
| **ERGs** | Community and belonging |

## Recovery Framework

### Individual Recovery

| Phase | Duration | Activities |
|-------|----------|-----------|
| **Stabilize** | 2-4 weeks | Reduce workload, rest, medical support |
| **Reflect** | 2-4 weeks | Identify root causes, set boundaries |
| **Rebuild** | 1-3 months | Gradual return, new habits, support system |
| **Sustain** | Ongoing | Regular check-ins, boundary maintenance |

### Organizational Recovery

| Action | Owner | Timeline |
|--------|-------|----------|
| Root cause analysis | Manager + HR | 1-2 weeks |
| Workload redistribution | Team | Immediate |
| Process changes | Leadership | 1 month |
| Culture changes | Whole org | 3-6 months |
| Follow-up surveys | HR | Quarterly |

## Anti-Patterns

| Anti-Patrón | Consecuencia |
|-------------|-------------|
| "Just take a vacation" | Treats symptom, not cause |
| Wellness perks without workload change | Tokenistic, breeds cynicism |
| Burning out the burnout responders | Helpers need help too |
| Individual therapy as only solution | Ignores systemic issues |
| "Hustle culture" glorification | Normalizes unsustainable pace |
