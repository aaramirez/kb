# Prácticas y Estándares

## Propósito

Esta sección define las prácticas técnicas y los estándares de calidad que todo equipo de desarrollo debe adoptar para mantener un código mantenible, testeable y seguro.

## Dimensiones clave

| Dimensión | Prácticas asociadas |
|-----------|-------------------|
| **Calidad de código** | Estándares, Clean Code, Refactoring |
| **Testing** | TDD, BDD |
| **Colaboración** | Pair Programming, Code Review |
| **Documentación** | DR, RFC, ADR |
| **Entrega** | Definition of Done, Feature Flags |
