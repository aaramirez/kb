# Cómo usar esta Knowledge Base

Esta base de conocimiento está diseñada para ser explorada de diversas formas según tus necesidades.

---

## Por Nivel de Experiencia

### Si eres nuevo en Management 3.0
1. Empieza por [[01-Fundamentos/README|Fundamentos]]
2. Continúa con [[02-Seis-Vistas/README|Las Seis Vistas]]
3. Explora [[03-Motivacion-Engagement/README|Motivación y Engagement]]
4. Revisa la [[mgmt3-timeline|Línea Temporal]]

### Si ya conoces los fundamentos
1. Profundiza en las prácticas: [[04-Delegacion-Empoderamiento/README|Delegación]], [[05-Objetivos-Metricas/README|Métricas]], [[06-Competencia-Habilidades/README|Competencia]]
2. Explora [[07-Estructura-Organizacional/README|Estructura]] y [[08-Gestion-Cambio/README|Cambio]]
3. Revisa [[09-Implementacion/README|Guía de Implementación]]

### Si estás implementando M3.0
1. Consulta [[09-Implementacion/README|Guía de Implementación]]
2. Revisa [[10-Ruta-Aprendizaje/README|Ruta de Aprendizaje]]
3. Usa el [[mgmt3-glossary|Glosario]] como referencia rápida

## Navegación

- Cada sección tiene un **README.md** como punto de entrada
- Los archivos usan **wikilinks** `[[...]]` para conectar conceptos relacionados
- El **Glosario** define todos los términos específicos
- La **Línea Temporal** muestra la evolución histórica

---

## Convenciones

- **Prefijo de tags**: `mgmt3/` para todas las notas
- **Numeración**: `01-`, `02-`, etc. para orden de lectura
- **Nomenclatura**: kebab-case en español
- **Fechas**: formato ISO (YYYY-MM-DD)
