## Corporate Huddles

Los Corporate Huddles son reuniones de sincronización regulares, cortas y enfocadas diseñadas para mantener al equipo **alineado, informado y conectado**. A diferencia de las reuniones tradicionales que suelen ser largas y poco productivas, los Huddles son breves, consistentes y orientados a la acción.

> **"Las mejores reuniones son las que la gente quiere atender, no las que tiene que atender."**

## Cuándo usarlos

- **Diario**: Sincronización rápida del equipo (standup)
- **Semanal**: Revisión de prioridades y bloqueos
- **Mensual**: Check-in más profundo sobre salud del equipo
- **Por proyecto**: Sincronización de stakeholders del proyecto
- **Post-incidente**: Reacción rápida a problemas

## Formato estándar

### Huddle diario (15 min máximo)

```
┌──────────────────────────────────────────────────────────────┐
│                    HUDDLE DIARIO                             │
│  Hora: 9:30 AM  Duración: 15 min  Frecuencia: Lun-Vie       │
│                                                              │
│  ┌──────────────────────────────────────────────────────┐   │
│  │ 1. ¿Qué hice ayer? (3 min)                          │   │
│  │    → Solo lo relevante para el equipo                 │   │
│  │                                                      │   │
│  │ 2. ¿Qué haré hoy? (3 min)                           │   │
│  │    → Compromisos claros                               │   │
│  │                                                      │   │
│  │ 3. ¿Qué me bloquea? (5 min)                         │   │
│  │    → Identificar impedimentos                         │   │
│  │                                                      │   │
│  │ 4. ¿Necesito ayuda? (4 min)                         │   │
│  │    → Ofertas y peticiones específicas                 │   │
│  └──────────────────────────────────────────────────────┘   │
│                                                              │
│  Reglas: De pie, sin teléfonos, a tiempo, Parking lot activo│
└──────────────────────────────────────────────────────────────┘
```

### Huddle semanal (30 min máximo)

| Fase | Duración | Contenido |
|------|----------|-----------|
| Check-in | 5 min | Cómo se siente cada uno (emoji o palabra) |
| Logros | 5 min | ¿Qué se logró esta semana? |
| Prioridades | 10 min | ¿Qué es lo más importante para la próxima semana? |
| Bloqueos | 5 min | ¿Qué impide avanzar? |
| Acciones | 5 min | ¿Quién hace qué antes del próximo huddle? |

### Huddle mensual (60 min máximo)

| Fase | Duración | Contenido |
|------|----------|-----------|
| Check-in profundo | 10 min | Cada persona comparte su estado (2 min c/u) |
| Métricas del mes | 10 min | Revisión de KPIs y tendencias |
| Celebraciones | 5 min | Reconocer logros del mes |
| Mejoras | 15 min | Identificar 1-2 mejoras para el próximo mes |
| Feedback | 10 min | Espacio para retroalimentación abierta |
| Siguiente mes | 10 min | Planificar el próximo ciclo |

## Reglas para Huddles efectivos

### Las 5 reglas de oro

| # | Regla | Por qué |
|---|-------|---------|
| 1 | **De pie** (o cámara encendida en remoto) | Mantiene la energía alta y la duración corta |
| 2 | **A tiempo** | Empezar y terminar cuando se dijo. Respetar el tiempo de todos |
| 3 | **Sin teléfonos** | Atención plena. Si es urgente, sal del huddle |
| 4 | **Parking lot** | Si un tema necesita más tiempo, anotarlo y tratarlo después |
| 5 | **Opcional pero consistente** | Asistencia no forzada, pero sí consistente |

### El facilitador

Cada huddle necesita un facilitador que:
- **Empiece a tiempo** — El huddle empieza cuando el primer llegado está listo
- **Mantenga el foco** — Derive al parking lot los temas que se desvían
- **Cierre con acuerdos** — Siempre terminar con "¿Quién hace qué antes de cuándo?"
- **Rote el rol** — Que todos faciliten al menos una vez al mes

## Anti-patrones

### Anti-patrones de contenido

| Anti-patrón | Problema | Solución |
|-------------|----------|----------|
| **Standup de estatus** | "Ayer hice X, hoy haré Y" sin valor | Enfocarse en bloqueos y necesidades |
| **Reunión de discusión** | Un tema domina todo el tiempo | Parking lot + reunión separada |
| **Comando y control** | El manager pregunta y todos responden | Que el equipo se sincronice entre sí |
| **Multitasking** | Gente trabajando en otras cosas | Regla de teléfonos y atención plena |
| **Reunión de 60 minutos** | Demasiado largo, pierde energía | Time-boxing estricto de 15 min |

### Anti-patrón de dinámica

| Anti-patrón | Problema | Solución |
|-------------|----------|----------|
| **El monólogo** | Una persona habla el 80% del tiempo | Ronda de turnos, facilitación activa |
| **La ausencia crónica** | Siempre falta alguien | Rotar horarios, grabar resumen |
| **El silencio** | Nadie habla, todos esperan | Facilitador pregunta directamente |
| **El optimismo infundado** | "Todo va bien" cuando no es así | Pedir datos, no percepciones |
| **La queja constante** | Siempre hay problemas, nunca soluciones | Reenfocar en "¿qué haremos?" |

## Huddles remotos

### Setup para equipos remotos

| Elemento | Recomendación |
|----------|---------------|
| **Plataforma** | Zoom, Teams, Google Meet — misma plataforma siempre |
| **Cámaras** | Encendidas obligatorias (salvo problemas de banda) |
| **Chat** | Usar para Parking lot, no para conversación paralela |
| **Pizarra digital** | Miro/Mural para visualizar acuerdos en tiempo real |
| **Grabación** | Opcional pero recomendado para ausentes |

### Tips para huddles remotos efectivos

1. **Empezar con check-in visual**: "¿Cómo está cada uno? (emoji o palabra)"
2. **Usar el chat para Parking lot**: Quien tenga un tema que se desvía, lo escribe ahí
3. **Rotar quién facilita**: Evitar que siempre sea la misma persona
4. **Cerrar con screenshot**: capturar los acuerdos y compartirlos
5. **Mantener la cámara encendida**: La conexión visual mantiene la atención

## Métricas de efectividad

| Métrica | Cómo medir | Objetivo |
|---------|------------|----------|
| **Asistencia** | % de asistencia promedio | >90% |
| **Puntualidad** | Minutos de retraso promedio | <2 min |
| **Parking lot** | Número de temas derivados | Decreciente |
| **Acciones completadas** | % de acuerdos cumplidos | >80% |
| **Satisfacción** | Encuesta mensual simple | >4/5 |

## Huddle vs Otras reuniones

| Aspecto | Huddle | Reunión tradicional |
|---------|--------|-------------------|
| Duración | 15-30 min | 60+ min |
| Frecuencia | Diaria/semanal | Según necesidad |
| Estructura | Fija, time-boxed | Variable, a menudo sin structure |
| Participación | Todo el equipo | Solo los "relevantes" |
| Salida | Acciones concretas | Minutas (a menudo ignoradas) |
| Cultura | Consistencia, brevedad | Ad-hoc, variable |
