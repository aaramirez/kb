# Evaluaciones 360 Grados

## ¿Qué es una evaluación 360?

Es un proceso de retroalimentación **multi-fuente** donde una persona recibe evaluación de todas las direcciones: gerentes, pares, reportes directos y auto-evaluación. Proporciona una visión completa del desempeño y comportamiento.

```
        Gerente
          │
    ┌─────┴─────┐
    │           │
  Pares ─── PERSONA ─── Reportes
    │           │
    └─────┬─────┘
          │
    Auto-evaluación
```

## Evaluación 360 en contexto de Management 3.0

M3.0 modifica el enfoque tradicional de 360:

| Tradicional | Management 3.0 |
|-------------|----------------|
| Evaluación anual | Ciclos más frecuentes (trimestral) |
| Calificaciones numéricas | Comentarios cualitativos |
| Vinculada a compensación | Vinculada a crecimiento |
| Enfocada en debilidades | Enfocada en fortalezas y contribuciones |
| Generada por RH | Facilitada por el equipo |

## Implementación paso a paso

### Fase 1: Preparación (1 semana)

1. **Definir el propósito**: ¿Para qué sirve esta evaluación? (Crecimiento, no castigo)
2. **Elegir participantes**: 4-6 personas que trabajen cercanamente con la persona
3. **Diseñar preguntas**: Enfocadas en comportamientos observables
4. **Comunicar**: Explicar el proceso, el anonymato y cómo se usará

### Fase 2: Recolección (1-2 semanas)

1. Cada participante responde de forma **anónima**
2. La persona también hace auto-evaluación
3. Se usan preguntas abiertas y escalas

#### Preguntas tipo

**Fortalezas:**
> "¿Cuál es la mayor contribución de esta persona al equipo?"
> "¿Qué deberían hacer MÁS esta persona?"

**Áreas de mejora:**
> "¿Qué debería hacer MENOS esta persona?"
> "Si pudieras cambiar UNA cosa, ¿cuál sería?"

**Comportamientos específicos:**
> "¿Esta persona facilita o bloquea la colaboración?"
> "¿Contribuye a un ambiente psicológicamente seguro?"

### Fase 3: Análisis (3-5 días)

1. Compilar respuestas (herramienta o manual)
2. Buscar patrones, no incidentes aislados
3. Identificar brechas entre auto-evaluación y percepción externa
4. Preparar un resumen visual

#### Plantilla de resultados

```
┌─────────────────────────────────────────────────┐
│        EVALUACIÓN 360 — [Nombre]               │
├─────────────────────────────────────────────────┤
│                                                 │
│  FORTALEZAS PERCIBIDAS:                         │
│  ✓ Comunicación clara (9/10 pares)             │
│  ✓ Colaboración (8/10 pares)                   │
│  ✓ Conocimiento técnico (7/10 pares)           │
│                                                 │
│  ÁREAS DE MEJORA:                               │
│  △ Delegación (solo 4/10 la ven fuerte)        │
│  △ Documentación (3/10 la ven fuerte)          │
│                                                 │
│  BRECHA AUTO-EVALUACIÓN vs EXTERNA:             │
│  Liderazgo: auto 8 vs externa 5                │
│                                                 │
└─────────────────────────────────────────────────┘
```

### Fase 4: Conversación (1 hora)

1. La persona revisa los resultados **antes** de la sesión
2. Un facilitador (no el gerente directo) guía la conversación
3. Se enfoca en:
   - "¿Qué te sorprendió?"
   - "¿Qué confirma lo que ya sabías?"
   - "¿Qué vas a hacer con esto?"
4. Se acuerdan 1-2 acciones concretas de desarrollo

### Fase 5: Seguimiento (continuo)

1. Las acciones se integran en el trabajo diario
2. Check-ins mensuales sobre el progreso
3. Próxima evaluación 360 en el próximo ciclo

## Errores comunes

| Error | Consecuencia | Solución |
|-------|-------------|----------|
| **Vinclar a compensación** | Respuestas infladas o inflamadas | Desvincular completamente |
| **Sin anonimato** | Respuestas diplomáticas, no honestas | Garantizar anonymato |
| **Sin follow-up** | Percepción de que no importa | Comprometerse con acciones |
| **Demasiadas preguntas** | Fatiga del encuestado | 5-8 preguntas máximo |
| **Solo identificar debilidades** | Desmoralización | Empezar por fortalezas |
| **Hacerla solo una vez** | No se ve progreso | Ciclos trimestrales |

## Frecuencia recomendada en M3.0

| Ciclo | Frecuencia | Propósito |
|-------|-----------|-----------|
| Rápido | Mensual (entre pares) | Feedback continuo informal |
| Estándar | Trimestral | Evaluación 360 formal ligera |
| Profundo | Semestral | Análisis de crecimiento a largo plazo |

## 360 como herramienta de equipo

Más allá de la evaluación individual, el 360 puede usarse para evaluar **al equipo como sistema**:

> "¿Qué tan bien funciona este equipo en las siguientes dimensiones?"
> - Comunicación
> - Colaboración
> - Toma de decisiones
> - Resolución de conflictos
> - Entrega de valor

Esto refuerza el principio de que el equipo, no el individuo, es la unidad de trabajo.
