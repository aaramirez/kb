# Matriz Comparativa de Herramientas

Esta matriz compara todas las herramientas Lean para ayudar a seleccionar la adecuada según el problema, el desperdicio que aborda, y la dificultad de implementación.

## Desperdicio vs Herramienta

Cada herramienta aborda uno o más [[03-Desperdicios|desperdicios]]:

| Desperdicio | Herramientas Principales | Herramientas Secundarias |
|------------|-------------------------|-------------------------|
| **Transporte** | Flujo Continuo, Manufactura Celular | VSM, 5S |
| **Inventario** | Kanban, Heijunka, Flujo Continuo | VSM, SMED |
| **Movimiento** | 5S, Standard Work, Manufactura Celular | Gemba |
| **Espera** | Flujo Continuo, Kanban | VSM, Cuellos de Botella |
| **Overproduction** | Kanban, Heijunka, Takt Time | Flujo Continuo |
| **Overprocessing** | Standard Work, Poka-Yoke | VSM, 5S |
| **Defectos** | Poka-Yoke, Standard Work, TPM | Andon, 5 Whys, A3 |
| **Sin uso de talento** | Hoshin Kanri, Kaizen | Standard Work, Gemba |

## Herramientas por Fase de Implementación

### Fase 1: Fundamentos (Meses 1-3)
| Herramienta | Propósito |
|------------|-----------|
| 5S | Organizar el espacio de trabajo |
| Gemba | Observar la realidad del proceso |
| PDCA | Motor de mejora continua |
| 5 Whys | Encontrar causas raíz |
| Takt Time | Entender el ritmo de demanda |
| Gestión Visual | Comunicar información |
| Kanban | Controlar inventario y flujo |

### Fase 2: Flujo y Señalización (Meses 3-6)
| Herramienta | Propósito |
|------------|-----------|
| Flujo Continuo | Eliminar inventario entre procesos |
| Standard Work | Estandarizar las mejores prácticas |
| Andon | Señalización de problemas |
| Poka-Yoke | Prevenir errores |
| VSM | Mapear y mejorar el flujo |
| Manufactura Celular | Organizar en celdas de flujo |

### Fase 3: Nivelación y Mantenimiento (Meses 6-12)
| Herramienta | Propósito |
|------------|-----------|
| Heijunka | Nivelar producción |
| SMED | Reducir tiempos de cambio |
| TPM | Mantenimiento productivo |
| OEE | Medir efectividad de equipos |
| Kaizen Event | Mejoras intensivas |
| A3 | Resolución estructurada |

### Fase 4: Estrategia y Cultura (Meses 12+)
| Herramienta | Propósito |
|------------|-----------|
| Hoshin Kanri | Alineación estratégica |
| Seis Grandes Pérdidas | Clasificación de pérdidas |
| RCA avanzado | Análisis complejo de causas |

| Combinación | Propósito |
|------------|-----------|
| 5S + Gemba | Base visual para observación |

| Flujo Continuo + Manufactura Celular | Unidades de flujo |
| PDCA + A3 | Estructura de mejora |
| 5 Whys + Ishikawa | Análisis de causa raíz |
| TPM + OEE + Seis Grandes Pérdidas | Mantenimiento y métricas |
| SMED + Heijunka | Cambios rápidos para nivelar |
| Poka-Yoke + Standard Work | Prevención + estandarización |
| VSM + Kaizen Event | Mapeo + acción rápida |
| Takt Time + Standard Work | Ritmo + secuencia |
