# Toyota Production System (TPS)

## Que es el TPS?

El Toyota Production System es el sistema de produccion desarrollado por Toyota entre 1948 y 1975. Es la base de todo el movimiento [[04-pensamiento-lean-womack-jones|Lean Thinking]] moderno.

El TPS se define como:

- Un sistema para detectar y resolver problemas relacionados con la calidad
- Un metodo para estandarizar las tareas
- Un sistema para hacer visibles los problemas
- La base del [[12-lean-vs-produccion-tradicional|Lean Manufacturing]]

## El Concepto de la Casa TPS

El [[14-casa-del-lean-detallada|TPS House]] es la representacion visual mas importante de este sistema. Sus elementos principales son:

### Techo: Objetivos

- **Calidad** - Cumplir con los requisitos del cliente a la primera
- **Costo** - Reducir desperdicios para minimizar costos
- **Entrega** - Tiempo de entrega confiable y corto
- **Seguridad** - Un entorno seguro para todas las personas
- **Moral** - Personal motivado y comprometido

### Pilares Fundamentales

El TPS se sustenta en dos pilares principales:

1. **JIT (Just-in-Time)** - Producir lo necesario, cuando se necesita, en la cantidad necesaria
2. **Jidoka** - Automatizacion con toque humano, detenerse ante anomalias

### Cimientos del Sistema

- **Heijunka** - Nivelacion de la produccion
- **[[05-cinco-principios-lean|Trabajo Estandarizado]]**
- **Kaizen** - Mejora continua

## JIT - Just-in-Time

El pilar JIT asegura que:

- Cada proceso produce solo lo que el siguiente proceso necesita
- Se elimina el [[06-valor|exceso de inventario]]
- El [[08-flujo|flujo continuo]] se mantiene
- El [[09-sistema-de-tiron|sistema de tiron]] regula la produccion

Componentes del JIT:

- Kanban (señal visual)
- Takt Time (ritmo de produccion)
- [[08-flujo|Flujo continuo]]
- [[09-sistema-de-tiron|Sistema de tiron]]

## Jidoka - Automatizacion con Toque Humano

Jidoka permite:

- Detectar anomalias en el proceso inmediatamente
- Detener la produccion para resolver problemas
- Prevenir la produccion de defectos
- Liberar a los trabajadores de tareas repetitivas
- Construir calidad en el proceso (no inspeccionar despues)

Conceptos relacionados con Jidoka:

- **Andon** - Sistema de alerta visual
- **Poka-Yoke** - A prueba de errores
- **Autonomacion** - Maquinas que se detienen solas

## Los Tres M: Muda, Mura, Muri

El TPS busca eliminar tres tipos de desperdicio:

### Muda (Desperdicio)

- Actividades que no agregan [[06-valor|valor]]
- Los 7 desperdicios clasicos de [[13-pensadores-lean-clave|Ohno]]
- Exceso de produccion, espera, transporte, procesamiento, inventario, movimiento, defectos

### Mura (Irregularidad)

- Variacion en los procesos
- Inconsistencia en los tiempos de ciclo
- Demanda fluctuante no nivelada

### Muri (Sobrecarga)

- Sobrecarga de las personas
- Sobrecarga de las maquinas
- Exigir mas de lo que el sistema puede entregar

## TPS vs Lean Manufacturing

Aunque relacionados, hay diferencias:

| Aspecto | TPS | Lean Manufacturing |
|---------|-----|-------------------|
| Origen | Toyota, Japon | Adaptacion global |
| Enfoque | Sistema completo | Filosofia + herramientas |
| Cultural | Japonesa | Internacional |
| Aplicacion | Primario manufactura | Multi-industria |

El TPS es la fuente; el Lean Manufacturing es la adaptacion global de sus principios. Ver [[12-lean-vs-produccion-tradicional]] para comparaciones detalladas.
