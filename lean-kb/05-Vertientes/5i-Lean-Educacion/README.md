# Lean Educación — Índice

La vertiente que adapta los principios del [[05-Vertientes/5a-Manufactura/01-lean-manufactura-vista-general|Lean Manufacturing]] al sector educativo. Lean Educación se enfoca en **maximizar el valor del aprendizaje** mientras se eliminan desperdicios en procesos administrativos y académicos.
