# Flujo Estudiantil (Student Flow)

## ¿Qué es el Flujo Estudiantil?

El Flujo Estudiantil es el recorrido completo que realiza un estudiante desde su interés inicial hasta la graduación. Mapear y optimizar este flujo es fundamental para eliminar [[01-Glosario/G-01-desperdicio|desperdicios]] como la espera y la fricción innecesaria.

> *"Cada barrera que enfrenta un estudiante es una oportunidad de mejora para la institución."*

## Mapeo del recorrido estudiantil

El mapeo del flujo estudiantil sigue estos pasos:

1. **Identificar etapas** — desde el interés inicial hasta la graduación
2. **Medir tiempos** — en cada punto de contacto
3. **Detectar cuellos de botella** — donde se acumulan barreras
4. **Visualizar el flujo** — con mapas de valor del estudiante
5. **Implementar mejoras** — reducir fricción y mejorar experiencia

### Etapas típicas del flujo

| Etapa | Actividad | Desperdicio común |
|-------|-----------|-------------------|
| **Interés** | Búsqueda de información | Falta de claridad |
| **Admisión** | Proceso de admisión | Burocracia excesiva |
| **Inscripción** | Registro y matrícula | Formularios redundantes |
| **Formación** | Asistencia a clases | Horarios rígidos |
| **Evaluación** | Exámenes y proyectos | Evaluación poco formativa |
| **Servicios** | Biblioteca, laboratorios | Acceso limitado |
| **Graduación** | Trámites de egreso | Proceso largo |

## Reducción de cuellos de botella

Los cuellos de botella más comunes en educación:

| Cuello de botella | Causa | Solución |
|-------------------|-------|----------|
| **Inscripción lenta** | Procesos manuales | Portal en línea |
| **Materias llenas** | Falta de planeación | Sistema de demanda |
| **Asesorías limitadas** | Recursos insuficientes | Tutorías entre pares |
| **Trámites de graduación** | Múltiples oficinas | Ventanilla única |
| **Biblioteca saturada** | Recursos físicos limitados | Recursos digitales |

## Optimización de la programación de cursos

La programación de cursos es un área crítica de optimización:

| Estrategia | Descripción | Beneficio |
|------------|-------------|-----------|
| **Demanda real** | Ofertar según demanda estudiantil | Mayor satisfacción |
| **Horarios flexibles** | Múltiples opciones de horario | Accesibilidad |
| **Cursos modulares** | Unidades independientes | Flexibilidad |
| **Online/híbrido** | Combinación presencial-virtual | Optimización de espacios |
| **Pre-requisitos claros** | Secuencia lógica de cursos | Progreso eficiente |

## Métricas de flujo estudiantil

| Métrica | Descripción | Objetivo |
|---------|-------------|----------|
| Tiempo de admisión | Días desde solicitud hasta respuesta | <5 días |
| Tasa de retención | % que continúa al siguiente semestre | >90% |
| Tiempo promedio de graduación | Semestres para completar carrera | Mínimo |
| Satisfacción estudiantil | Encuesta de experiencia | >85% |
| Uso de servicios | % de estudiantes que usan servicios | Maximizar |
