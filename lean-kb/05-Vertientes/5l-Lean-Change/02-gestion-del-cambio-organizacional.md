# Gestión del Cambio Organizacional

## Fundamentos del Change Management

La gestión del cambio es el conjunto de procesos, herramientas y técnicas para gestionar el **lado humano** de los cambios en una organización.

## Modelo de Kotter y Lean

John Kotter definió 8 pasos para el cambio. Veamos cómo Lean los complementa:

| Paso Kotter | Enfoque Lean |
|-------------|--------------|
| 1. Crear urgencia | Datos de desperdicio (muda) |
| 2. Formar coalición | Liderazgo Lean distribuido |
| 3. Crear visión | Value Stream futuro |
| 4. Comunicar visión | Visual management del cambio |
| 5. Empoderar acción | Kaizen events |
| 6. Generar victorias | Mejoras rápidas visibles |
| 7. Consolidar | Estandarizar cambios exitosos |
| 8. Anclar en cultura | Kaizen como hábito |

## Modelo ADKAR

ADKAR (Awareness, Desire, Knowledge, Ability, Reinforcement) conecta directamente con Lean:

- **Awareness**: entender el desperdicio actual (Gemba walk)
- **Desire**: crear motivación mediante participación
- **Knowledge**: capacitación en nuevas prácticas
- **Ability**: práctica guiada en el Gemba
- **Reinforcement**: estandarización y reconocimiento

## Gestión de la resistencia

La resistencia no es un obstáculo; es **información valiosa**:

```
Resistencia = "Algo no está claro"
             = "No veo el beneficio"
             = "Tengo miedo de perder algo"
             = "No fui consultado"
```

Cada tipo de resistencia requiere una respuesta diferente.

## El rol del liderazgo Lean en el cambio

- **Ir al Gemba**: ver el cambio happening
- **Preguntar, no decir**: facilitar, no imponer
- **Modelar el cambio**: ser el primero en adoptar
- **Celebrar aprendizajes**, no solo éxitos
