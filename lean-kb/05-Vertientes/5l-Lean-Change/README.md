# Lean Change

## Vista General

La vertiente **Lean Change** aplica los principios Lean a la gestión del cambio organizacional. El cambio se trata como un **proceso a optimizar**, no como un evento puntual. Se enfoca en experimentos, feedback loops y respeto por las personas.
