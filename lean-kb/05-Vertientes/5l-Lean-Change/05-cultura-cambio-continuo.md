# Cultura de Cambio Continuo

## Construir una organización que abrace el cambio

Una cultura de cambio continuo no es un proyecto con fecha de fin; es una **forma de ser**. Lean y Kaizen son la base natural para esto.

## Kaizen culture como change culture

Kaizen y cambio continuo son esencialmente lo mismo:

```
Kaizen = Mejora continua = Cambio continuo = Adaptación constante
```

Las organizaciones con cultura Lean ya tienen el músculo del cambio; solo necesitan expandirlo.

## Comportamientos de liderazgo

### Lo que los líderes Lean hacen

| Comportamiento | Impacto |
|----------------|---------|
| **Modelar el cambio** | "Yo también estoy aprendiendo" |
| **Admitir errores** | "Este experimento no funcionó" |
| **Preguntar, no decir** | "¿Qué sugieres?" |
| **Celebrar aprendizajes** | "¿Qué descubrimos?" |
| **Dar autonomía** | "Tú decides cómo implementar" |

### Lo que los líderes Lean evitan

- Imponer cambios sin consultar
- Castigar experimentos fallidos
- Exigir perfección desde el inicio
- Ignorar el feedback del Gemba

## Manteniendo el cambio a largo plazo

### Los 3 pilares de sustentabilidad

1. **Estandarización**: documentar lo que funciona
2. **Medición**: métricas que muestran impacto
3. **Reconocimiento**: celebrar y compartir éxitos

### Evitar la fatiga del cambio

- No lanzar demasiados cambios simultáneamente
- Priorizar por impacto y esfuerzo
- Permitir tiempo de absorción
- Rotar los esfuerzos de cambio entre equipos

## Señales de una cultura de cambio saludable

- El equipo **propone** mejoras sin que se les pida
- Los **experimentos** fallidos se celebran como aprendizaje
- El **Gemba walk** es rutina, no evento
- La **retrospectiva** genera acciones concretas
- El **cambio** se siente como parte del trabajo, no como extra

## Del proyecto a la cultura

```
Proyecto de cambio:     Cultura de cambio:
Tiene inicio y fin      Es continua
Tiene un dueño          Es de todos
Tiene un plan fijo      Se adapta
Se mide al final        Se mide siempre
```
