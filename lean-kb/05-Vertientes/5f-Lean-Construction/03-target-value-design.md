# Target Value Design (TVD)

## ¿Qué es Target Value Design?

Target Value Design (TVD) es una metodología de diseño que parte del **costo objetivo** definido por el cliente y trabaja hacia atrás para lograr ese costo sin sacrificar la calidad. Es el equivalente al diseño de costo en manufactura aplicado a la construcción.

> *"El presupuesto no es el resultado del diseño, es el punto de partida."*

## Diseño dirigido por costo (Design-to-Cost)

A diferencia del enfoque tradicional donde se diseña y luego se calcula el costo, TVD invierte la secuencia:

### Enfoque tradicional
1. Diseñar el proyecto
2. Calcular el costo estimado
3. Negociar reducciones si excede el presupuesto
4. Comprometer calidad o funcionalidad

### Enfoque TVD
1. Definir el costo objetivo con el cliente
2. Diseñar dentro de ese costo
3. Innovar para lograr funcionalidad dentro del presupuesto
4. Optimizar valor sin exceder el costo objetivo

## Colaboración multifuncional

TVD requiere la participación temprana de todos los actores clave:

| Rol | Contribución | Momento |
|-----|-------------|---------|
| **Cliente/Propietario** | Define el valor y el costo objetivo | Inicio del proyecto |
| **Diseñador** | Soluciones técnicas dentro del costo | Fase de diseño |
| **Contratista** | Factibilidad constructiva y costos reales | Fase de diseño |
| **Subcontratistas** | Especialización y costos específicos | Fase de diseño |
| **Proveedores** | Materiales y soluciones innovadoras | Fase de diseño |

## Valor vs costo

El objetivo de TVD es **maximizar el valor dentro del costo objetivo**:

```
Valor = Funcionalidad / Costo
```

### Estrategias para maximizar valor
- **Simplificar el diseño** — eliminar componentes no esenciales
- **Innovar en materiales** — soluciones más económicas pero equivalentes
- **Optimizar procesos** — reducir costos de construcción
- **Diseñar para fabricabilidad** — facilitar la ejecución
- **Eliminar sobre-diseño** — remover especificaciones excesivas

## Relación con LPS

TVD funciona como complemento del [[05-Vertientes/5f-Lean-Construction/02-last-planner-system|Last Planner System]]:

- **TVD** define el QUÉ se construye y a QUÉ costo
- **LPS** define CÓMO y CUÁNDO se construye
- Juntos aseguran que el proyecto se entregue dentro del valor y costo objetivo

## Métricas de TVD

| Métrica | Descripción | Objetivo |
|---------|-------------|----------|
| Adherencia al costo | Costo real vs objetivo | Dentro del 5-10% |
| Valor percibido | Satisfacción del cliente | Máximo |
| Cambios de diseño | Variaciones durante el proyecto | Minimizar |
| Eficiencia de diseño | Tiempo de diseño vs estándar | Mejorar |
