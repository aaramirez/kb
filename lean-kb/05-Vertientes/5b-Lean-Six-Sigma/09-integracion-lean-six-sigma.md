# Integración Lean y Six Sigma

## ¿Por qué integrar?

Lean y Six Sigma no compiten — se **complementan**. Lean ataca la velocidad y el flujo, Six Sigma ataca la variabilidad y los defectos. Juntos cubren más problemas que por separado.

> *"Lean te hace rápido. Six Sigma te hace preciso. Juntos te hacen imparable."*

## Complementariedad

| Problema | Lean | Six Sigma | LSS integrado |
|----------|------|-----------|---------------|
| Demasiado inventario | ✅ Pull, kanban | ⚠️ Parcial | ✅ Reducir variabilidad + pull |
| Defectos altos | ⚠️ Jidoka | ✅ SPC, root cause | ✅ Flujo + calidad |
| Lead time largo | ✅ Flujo continuo | ⚠️ Parcial | ✅ Optimizar + estabilizar |
| Procesos inestables | ⚠️ Estandarizar | ✅ Control estadístico | ✅ Estandarizar + monitorear |
| Sobreproducción | ✅ Pull | ⚠️ Parcial | ✅ Demanda + capacidad |

## Cuándo usar cada enfoque

### Usar Lean cuando:
- El problema es **flujo** y **desperdicio**
- Se necesita resultado rápido
- Hay poco acceso a datos estadísticos
- El equipo es nuevo en mejora de procesos

### Usar Six Sigma cuando:
- El problema es **variabilidad** y **calidad**
- Hay datos históricos disponibles
- Se requiere rigor estadístico
- El problema es complejo y multifactorial

### Usar LSS cuando:
- El problema tiene **ambos** componentes
- La organización tiene cultura de datos
- Se busca impacto financiero medible
- El proceso requiere estabilidad Y flujo

```
Kaizen               +   Hypothesis Testing
```

## Casos de integración

### Caso 1 — Reducir defectos en línea de ensamble
1. **Lean:** Mapear cadena de valor, identificar desperdicios
2. **Six Sigma:** Analizar causas de defectos con datos
3. **Resultado:** 40% menos defectos, 30% menos lead time

### Caso 2 — Reducir inventario
1. **Six Sigma:** Estabilizar variabilidad de demanda
2. **Lean:** Implementar pull y kanban
3. **Resultado:** 60% menos inventario, misma disponibilidad

### Caso 3 — Mejorar entrega
1. **Lean:** Eliminar transporte y espera innecesaria
2. **Six Sigma:** Reducir variación en tiempos de proceso
3. **Resultado:** 95% on-time delivery (era 78%)
