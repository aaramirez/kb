# Moderación y Dinámica

## Rol del facilitador

El facilitador de Lean Coffee no es un **orador** ni un **experto**; es un **guía** que mantiene el proceso funcionando.

### Responsabilidades

- Mantener el **ritmo** del tiempo
- Asegurar que **todos participen**
- Proteger el **espacio seguro** para hablar
- **No** dominar la discusión con opiniones propias
- Ejecutar los **votos** de Thumbs up/down

## Técnicas de gestión del tiempo

### Temporizador visible

Siempre visible para todos. Cuando suena, se respeta.

```
0:00 ─── 1:00 ─── 5:00 ─── 10:00
 │        │        │         │
Primer   Voto    Extensión  Siguiente
voto     U/D     (si hay)   tema
```

### Técnica del reloj

- Asignar un **reloj** a cada tema
- Cuando el reloj suena, se vota
- Si se continúa, se resetea

### Buffer de tiempo

Reservar los últimos 5 minutos para retrospectiva. No consumirlos en discusión.

## Mantener discusiones enfocadas

### Señales de que la discusión se desvía

- Se repiten los mismos argumentos
- Se habla de temas no relacionados
- Solo 2 personas hablan
- El tono se vuelve emocional

### Acciones del facilitador

| Situación | Acción |
|-----------|--------|
| Repetición | "¿Hay algo nuevo que agregar?" |
| Desviación | "Esto es interesante, pero fuera de scope" |
| Dominancia | "¿Qué opinan otros?" |
| Silencio | "¿Alguien que no ha hablado?" |
| Conflicto | "Ambos puntos son válidos, pasemos" |

## Manejo de participantes dominantes

1. **Reconocer** su contribución
2. **Rotar** el tiempo de palabra
3. **Preguntar** directamente a otros
4. **Usar** la regla de "una idea por turno"

## Asegurar participación igualitaria

- **Brainwriting** antes de discutir (todos contribuyen por escrito)
- **Dot voting** (todos tienen los mismos puntos)
- **Rotación** de quién habla primero
- **Silencio** al inicio para reflexionar

## Dinámica para grupos grandes (+8 personas)

- Dividir en **sub-grupos** de 4-5
- Cada sub-grupo tiene su tablero
- Al final, **compartir** los temas más votados
- Un facilitador por sub-grupo
