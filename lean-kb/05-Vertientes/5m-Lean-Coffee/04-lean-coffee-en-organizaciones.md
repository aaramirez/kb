# Lean Coffee en Organizaciones

## Aplicación corporativa

Lean Coffee no es solo para comunidades externas; es una herramienta poderosa dentro de las organizaciones.

## Team building y knowledge sharing

### Beneficios organizacionales

- **Rompe silos**: personas de diferentes áreas conversan
- **Comparte conocimiento**: lo que uno sabe, otros aprenden
- **Construye relaciones**: el formato informal facilita conexiones
- **Identifica expertos**: quién sabe qué se hace visible

### Frecuencia recomendada

- **Semanal**: para equipos pequeños (30 min)
- **Quincenal**: para departamentos (45 min)
- **Mensual**: para organizaciones (60 min)

## Formato de Innovation Workshop

Usar Lean Coffee como herramienta de innovación:

```
┌──────────────────────────────────────┐
│  Innovation Lean Coffee              │
│                                      │
│  1. Generar ideas (brainwriting)     │
│  2. Votar las más prometedoras       │
│  3. Discutir viabilidad (5 min c/u) │
│  4. Seleccionar 2-3 para prototipar  │
│  5. Asignar owners y siguiente paso  │
└──────────────────────────────────────┘
```

## Retrospetiva alternativa

Lean Coffee puede reemplazar o complementar la retrospectiva tradicional:

| Retrospectiva tradicional | Lean Coffee Retro |
|---------------------------|-------------------|
| Facilitador prepara preguntas | Equipo genera temas |
| Mismo formato siempre | Formato flexible |
| Puede ser predecible | Siempre emergente |
| 1-2 horas | 30-45 minutos |

### Formato Lean Coffee Retro

1. Cada uno escribe: "¿Qué debería mejorar?" (sticky notes)
2. Votar los temas
3. Discutir soluciones con timeboxing
4. Definir acciones concretas

## Lean Coffee Virtual

Para equipos remotos o distribuidos:

### Herramientas digitales

| Herramienta | Uso |
|-------------|-----|
| **Miro** | Tablero virtual con sticky notes |
| **Trello** | Tablero Kanban para temas |
| **Slack** | Canal dedicado + polls |
| **Zoom/Teams** | Video con breakout rooms |
| **FunRetro** | Retrospectivas distribuidas |

### Tips para virtual

- Usar **cámara encendida** para engagement
- **Chat habilitado** para contribuciones paralelas
- **Temporizador compartido** en pantalla
- **Breakout rooms** para grupos grandes

## Retos en organizaciones

- **Cultura jerárquica**: puede ser difícil que todos hablen
- **Tiempo**: "no tenemos 30 minutos extra"
- **Resultados**: medir el impacto para justificar continuidad
- **Escalamiento**: mantener la intimidad en grupos grandes
