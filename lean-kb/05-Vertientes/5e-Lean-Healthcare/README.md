# Lean Healthcare — Índice

La vertiente que aplica los principios del [[05-Vertientes/5a-Manufactura/01-lean-manufactura-vista-general|Lean Manufacturing]] al sector salud. Lean Healthcare se enfoca en **maximizar el valor para el paciente** eliminando desperdicios en procesos clínicos y administrativos, mejorando tiempos de espera y elevando la calidad de la atención.
