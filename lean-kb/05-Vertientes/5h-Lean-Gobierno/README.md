# Lean Gobierno — Índice

La vertiente que adapta los principios del [[05-Vertientes/5a-Manufactura/01-lean-manufactura-vista-general|Lean Manufacturing]] al sector público. Lean Gobierno se enfoca en **mejorar la calidad de los servicios gubernamentales** mientras se eliminan desperdicios burocráticos y se reduce la complejidad innecesaria.
