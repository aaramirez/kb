# Vertientes de Lean - Vista General

## Las 13 ramas del pensamiento Lean

Lean no es un sistema único y rígido. Ha evolucionado en múltiples **vertientes** adaptándose a diferentes industrias, contextos y necesidades. Cada vertiente preserva los principios centrales pero los aplica de forma específica.

## Pollinización cruzada

Las vertientes no son compartimentos estancos. Lo mejor de una puede aplicarse a otra:

- **Kanban** de Lean Software → se usa en Lean Healthcare
- **Gemba Walk** de Manufacturing → se aplica en Lean Government
- **PDCA** de Lean Manufacturing → base de Lean Startup
- **Visual Management** de Lean Coffee → herramienta en cualquier vertiente
- **Respeto por las personas** → universal a todas las vertientes
