# Agile-Lean

## Vista General

La vertiente **Agile-Lean** explora la convergencia entre las metodologías ágiles y los principios Lean. Agile nació del Lean pero ha evolucionado para enfocarse específicamente en el desarrollo de software y productos digitales, mientras que Lean mantiene su visión sistémica de optimización de flujos de valor.
