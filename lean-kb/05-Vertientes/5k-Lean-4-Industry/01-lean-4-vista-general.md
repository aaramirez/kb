# Lean 4.0 - Vista General

## ¿Qué es Lean 4.0?

**Lean 4.0** es la evolución del pensamiento Lean que integra las tecnologías de la **Cuarta Revolución Industrial** (Industria 4.0) con los principios clásicos de eliminación de desperdicio y mejora continua.

No reemplaza Lean; lo **potencia** con datos, automatización e inteligencia.

## Tecnologías clave

| Tecnología | Aplicación Lean |
|------------|-----------------|
| **Digital Twins** | Simulación de value streams |
| **IoT** | Detección en tiempo real de anomalías |
| **AI/ML** | Análisis predictivo y prescriptivo |
| **Big Data** | Análisis de causa raíz a escala |
| **Robótica** | Automatización de tareas repetitivas |
| **Blockchain** | Trazabilidad en cadena de valor |

## Beneficios de Lean 4.0

```
Lean Clásico          Lean 4.0
─────────────         ──────────────
Observación visual    → Detección IoT automática
Kaizen manual         → AI-powered improvement
Pull manual           → Smart pull systems
VSM en papel          → Digital twins dinámicos
Métricas periódicas   → Métricas en tiempo real
```

## Los 8 desperdicios en la era digital

Los desperdicios de Lean se expanden:

1. **Overprocessing** → Procesamiento innecesario de datos
2. **Waiting** → Latencia en sistemas
3. **Defectos** → Bugs y datos incorrectos
4. **Motion** → Navegación innecesaria entre sistemas
5. **Inventory** → Datos almacenados sin usar
6. **Transportation** → Copias innecesarias de datos
7. **Overproduction** → Reportes no solicitados
8. **Non-utilized talent** → Habilidades digitales subutilizadas

## Principios de Lean 4.0

1. **Datos como base de decisiones** — no intuición
2. **Automatización inteligente** — no automatizar desperdicio
3. **Conectividad total** — visibilidad end-to-end
4. **Mejora continua acelerada** — ciclos más cortos con más datos
5. **Respeto por las personas** — la tecnología complementa, no reemplaza
