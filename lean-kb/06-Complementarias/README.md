# Metodologías Complementarias

Sección dedicada a las metodologías que **complementan y potencian** [[Lean Manufacturing]], aportando herramientas estadísticas, estructura de resolución de problemas y gestión de calidad que Lean no cubre de forma autónoma.

## Visión General

Lean por sí solo es poderoso, pero tiene limitaciones en áreas como el análisis estadístico profundo, la gestión formal de calidad y la resolución estructurada de problemas complejos. Las metodologías complementarias llenan estos vacíos.

> *"Lean nos da la velocidad. Las complementarias nos dan la precisión."*

## Guía de Decisión: ¿Cuándo Usar Cada Una?

| Situación | Metodología Recomendada | Nota |
|-----------|------------------------|------|
| El proceso tiene mucha variabilidad | [[01-six-sigma-vista-general\|Six Sigma]] | Reducir variación con datos |
| Hay un cuello de botella claro | [[02-teoria-de-restricciones\|TOC]] | Enfocarse en la restricción |
| Se necesita cultura de calidad general | [[03-tqm-gestion-calidad-total\|TQM]] | Filosofía organizacional |
| Problema de calidad específico y urgente | [[04-8d-resolucion-problemas\|8D]] | Resolución en equipo |
| Evaluar riesgos de diseño o proceso | [[05-fmea-analisis-modos-falla\|FMEA]] | Análisis preventivo |
| Monitorear estabilidad de procesos | [[06-spc-control-estadistico\|SPC]] | Control con gráficas |
| Optimizar parámetros de proceso | [[07-doe-diseno-experimentos\|DOE]] | Diseño de experimentos |

## Combinaciones Populares

- **Lean Six Sigma** — Lean + Six Sigma para velocidad y calidad
- **Lean TOC** — Lean + Teoría de Restricciones para flujo óptimo
- **TQM + Lean** — Calidad como cultura con herramientas de flujo
- **8D + FMEA** — Resolución reactiva + análisis preventivo
