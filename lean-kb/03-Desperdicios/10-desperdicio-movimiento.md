# Desperdicio: Movimiento

**Movimiento** se refiere al desplazamiento innecesario de personas dentro de un proceso. A diferencia de [[07-desperdicio-transporte|transporte]] (materiales), este desperdicio afecta directamente a los operadores.

## Definición

El movimiento es desperdicio cuando:
- El operador camina para buscar herramientas
- Se giran o inclinan el cuerpo innecesariamente
- Se alcanzan materiales fuera del área de trabajo
- Se cambia de posición para acceder a información
- Se repite un movimiento que podría automatizarse

## Causas Comunes

### Diseño de Estación de Trabajo
Herramientas y materiales no están en posiciones ergonómicas de fácil acceso.

### Falta de 5S
El área de trabajo no está organizada, ordenada y estandarizada.

### Layout Ineficiente
Los procesos adyacentes están demasiado separados.

### Ausencia de Estandarización
Cada operador organiza su espacio de forma diferente.

## Impacto

El movimiento innecesario causa:

- **Fatiga** — Desgaste físico del operador
- **Pérdida de tiempo** — Tiempo no productivo
- **Riesgo de lesiones** — Movimientos repetitivos o forzados
- **Estrés** — Sensación de no poder completar el trabajo cómodamente

## Movimientos Desperdiciados

Según los principios de [[Ergonomía]] y análisis de movimiento:

| Tipo | Ejemplo |
|------|---------|
| Alcanzar | Estirar el brazo para tomar una herramienta lejana |
| Buscar | Localizar un archivo o herramienta |
| Girar | Voltear el cuerpo para acceder al siguiente paso |
| Inclinarse | Agacharse para tomar materiales |
| Caminar | Desplazarse entre áreas de trabajo |
| Levantar | Elevar cargas que requieren ayuda mecánica |

## Contraerrestantes

### 5S
Organizar el área de trabajo para que todo esté en su lugar y sea fácil de encontrar.

### Diseño Ergonómico
Ubicar herramientas y materiales dentro del arco de movimiento natural.

### Trabajo Estandarizado (Standard Work)
Definir la secuencia óptima de movimientos para cada tarea.

### Automatización
Eliminar movimientos repetitivos mediante automatización.

## Ejemplo

Un operador gira 180° para tomar piezas de un contenedor detrás de él. Colocando el contenedor a su derecha, dentro del alcance natural, se elimina el giro innecesario y se reduce la fatiga.

## Diferencia con Transporte

No confundir con [[07-desperdicio-transporte|transporte]], que se refiere al desplazamiento de **materiales o productos**. Movimiento se refiere al desplazamiento de **personas**.
