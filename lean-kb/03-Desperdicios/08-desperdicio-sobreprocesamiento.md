# Desperdicio: Sobreprocesamiento

**Sobreprocesamiento** es realizar más trabajo del necesario para satisfacer las necesidades del cliente. Incluye pasos redundantes, procesos que no agregan valor y requisitos excesivos.

## Definición

Sobreprocesamiento ocurre cuando:
- Se realizan más pasos de los requeridos
- Se utiliza mayor precisión de la necesaria
- Se generan reportes que nadie consulta
- Se aprueban documentos múltiples veces
- Se reprocesa trabajo ya completado

## Causas Comunes

### Falta de Estandarización
Cada persona ejecuta el proceso a su manera, incluyendo pasos innecesarios.

### Diseño de Producto Complejo
Diseños con especificaciones excesivas que requieren procesamiento adicional.

### Falta de Confianza
Doble verificación, inspecciones redundantes y aprobaciones innecesarias.

### Herramientas Obsoletas
Equipos que requieren trabajo manual adicional que podría evitarse.

## Ejemplos Comunes

| Actividad | Descripción |
|-----------|-------------|
| Reportes innecesarios | Generar informes que no se utilizan para decisiones |
| Aprobaciones múltiples | Mismo documento revisado por 5 personas |
| Acabado excesivo | Pulir superficies que no son visibles al cliente |
| Formato excesivo | Requiere 10 copias de un mismo documento |
| Mediciones redundantes | Inspeccionar lo que ya fue verificado |

## Contraerrestantes

### Trabajo Estandarizado (Standard Work)
Documentar y seguir el proceso mínimo necesario para entregar calidad.

### Análisis de Valor (Value Analysis)
Evaluar cada paso del proceso preguntando si agrega valor.

### Diseño para Manufactura (DFM)
Diseñar productos que requieran procesamiento mínimo.

### Autonomación (Jidoka)
Detener el proceso cuando algo no está bien, en lugar de seguir procesando.

## Ejemplo

Un departamento genera un reporte semanal de 20 páginas. Solo se utilizan 3 gráficos para tomar decisiones. Simplificar el reporte a esas 3 páginas elimina sobreprocesamiento sin perder información útil.
